# grpc

grpc是谷歌发起的一种rpc实现,其基本特点是:

+ 使用`http2`协议传输数据
+ 使用[protobuf]序列化数据
+ 使用`protobuf`定义接口
+ 双工服务
+ 主流编程语言支持
+ 健全生态

数据预测一般都需要落地为服务,grpc凭借出色的性能和健全的生态常常是最优的选择.
